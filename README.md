# Welcome to Poketeams!

The purpose of this project is to build something fun with a
stack I haven't used before. That being the framework version of
React Router v7. It handles both frontend and backend routing
with some interesting advantages that I'd like to get more familiar with.

## Tech stack

React Router v7 (SSR, framework mode) · Drizzle ORM · PostgreSQL · TailwindCSS v4 · Zod · TypeScript
